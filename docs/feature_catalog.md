# Feature Catalog

Feature namespaces:

- `weather_`: ERA5-Land aggregates, VPD, wind speed, and drought proxies.
- `optical_`: Sentinel-2 band summaries and spectral indices.
- `terrain_`: elevation, slope, TWI, and terrain derivatives.
- `temporal_`: historical fire counts and temporal deltas.
- `landcover_`: Dynamic World fractions.
- `anthropogenic_`: distances to roads, rivers, and settlements.
- `peat_`: peat fraction and related static wetland attributes.
- `human_`: population density features.
